## 电梯助手APP项目简介

本项目是基于 **uni-app + Vue 2 + uView UI** 的多端应用前端工程，支持构建到 H5、App、各类小程序及快应用等平台，用于"梯递助手/友惠见"等相关业务场景的客户端展示与交互。

## 主要功能

### 🏠 首页功能
- **轮播图展示**：支持首页轮播广告展示
- **消息滚动栏**：系统通知滚动展示
- **宫格菜单**：快速入口（合伙人、个人中心、福利中心、优惠券、活动中心、邀请好友、任务中心、我的资产）
- **商品展示**：热门商品列表展示
- **任务展示**：热门任务列表展示
- **扫一扫**：二维码扫描功能

### 👤 用户功能
- **登录注册**：支持手机号登录、社交登录（微信授权）
- **个人资料**：用户信息管理
- **我的订单**：订单状态管理（待支付、待发货、待收货、已完成）
- **我的团队**：团队信息查看
- **二维码管理**：个人二维码、公司二维码生成与分享
- **收货地址**：地址的增删改查
- **资产信息**：NFT等级、现金余额、积分、碳值、保证金查看

### 📋 任务功能
- **任务列表**：查看所有可用任务
- **任务详情**：任务详细信息和完成要求
- **任务完成**：提交任务完成结果，获得佣金奖励

### 🛒 商品功能
- **商品分类**：商品分类浏览
- **商品列表**：分类下的商品列表
- **商品详情**：商品详细信息、价格、规格
- **购物车**：商品加购管理

### 🏢 电梯管理功能
- **扫码乘梯**：扫描二维码进入电梯调度页面
- **楼层选择**：选择目标楼层
- **实时调度**：通过 MQTT 协议实时接收电梯调度状态
- **调度状态**：显示电梯调度中、运行中、到达等状态
- **邀请码验证**：临时访客需输入邀请码

### 👥 合伙人功能
- **合伙人申请**：申请成为城市合伙人、省总督
- **团队收益**：查看团队收益统计
- **合伙人管理**：合伙人相关功能

### 🔧 管理员功能（需权限）
- **人员管理**：员工信息审核与管理
- **楼栋管理**：楼栋信息的增删改查
- **公司管理**：公司信息的增删改查
- **电梯管理**：电梯设备信息管理
- **电梯广告机管理**：电梯内广告机设备管理
- **大厅广告机管理**：大厅广告机设备管理

### 🔍 其他功能
- **搜索功能**：商品、任务搜索
- **支付功能**：签约收银台、支付页面
- **分享功能**：支持微信分享、H5分享、APP分享
- **下载页**：APP下载引导页
- **引导页**：应用启动引导

## 开发环境要求

- **Node.js**：建议 14.x 或 16.x（保持与当前环境一致即可）
- **包管理工具**：`yarn` 或 `npm`
- **HBuilderX / VS Code** 等 IDE（推荐 VS Code + uni-app 插件）

## 安装依赖

使用 `yarn`（推荐）：

```bash
yarn
```

或使用 `npm`：

```bash
npm install
```

## 本地运行（开发环境）

H5 端预览：

```bash
yarn serve
# 或
npm run serve
```

运行成功后，根据终端输出的地址，在浏览器中访问即可进行调试。

如果需要针对具体平台进行开发调试，也可以使用：

```bash
# H5
yarn dev:h5

# App（需配合 HBuilderX 或真机调试）
yarn dev:app-plus

# 微信小程序
yarn dev:mp-weixin
```

## 打包构建

常用打包命令（根据需要选择执行）：

```bash
# H5 正式环境打包
yarn build

# H5 自定义环境打包（示例）
yarn build:h5

# App 正式打包
yarn build:app-plus

# 微信小程序打包
yarn build:mp-weixin
```

项目中还提供了多种平台及多种输出目录的构建脚本，例如：

- `build-product`：输出到 `dist/build/h5pack/pro`
- `build-development`：输出到 `dist/build/h5pack/dev`
- `build-local`：输出到 `dist/build/h5pack/local`

可根据实际部署流程选择对应命令。

## 项目结构简介

关键目录说明（部分）：

- `src/pages`：业务页面（登录、任务、个人中心、广告、电梯等）
- `src/components`：通用组件（二维码扫描、楼层选择、倒计时、选择器等）
- `src/store`：`vuex` 状态管理
- `src/api`：接口封装（用户、任务、商品、广告、电梯、地址等）
- `src/utils/request`：请求封装与拦截器
- `src/uni_modules`：第三方 uni-app 组件（计时器、验证码等）

## 常见问题

- **依赖安装失败**：请检查 Node 版本是否过低或网络是否有代理限制；可尝试删除 `node_modules` 与 `yarn.lock`/`package-lock.json` 后重新安装。
- **端口占用**：本地运行时如端口被占用，可在 `vue.config.js` 或启动参数中进行端口调整，或关闭占用端口的进程后重试。
